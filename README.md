# open-google
> Open a Google search from node

## Installation
```
$ npm install open-google
```

## Usage
```js
const openGoogle = require('open-google');

openGoogle('Knut Kirkhorn');
```

## API
### openGoogle(search)
Opens the webpage for the Google result of the search.
